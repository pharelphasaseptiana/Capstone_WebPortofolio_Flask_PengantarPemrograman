## Deskripsi

Aplikasi portofolio berbasis Flask yang dilengkapi dashboard admin untuk mengelola profil, proyek, dan pesan dari pengunjung.

## Fitur

- Halaman Beranda
- Halaman Tentang
- Halaman Portofolio
- Detail Proyek
- Form Kontak
- Dashboard Admin
- CRUD Proyek
- CRUD Profil
- Manajemen Pesan

## Teknologi

- Python
- Flask
- SQLAlchemy
- SQLite / PostgreSQL
- Bootstrap
- HTML
- CSS
- JavaScript
